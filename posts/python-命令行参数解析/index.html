位置参数

  • 添加了 add_argument() 方法，该方法用来指定程序接受哪些命令行选项。现在程序需要我们指定一个选项。
  • parse_args() 方法从指定的选项返回一些数据，在这里是 echo 。
  • args 变量中存储的选项值与添加选项时的字符串是匹配的，都是"echo"。

usage: ooo.py [-h] [--verbose]

options:
  -h, --help  show this help message and exit
  --verbose   increase output verbosity
  • 与前面不同，这里的选项是一个标志量而不是需要一个值的变量。同时也更改了选项的名字来适配这个想法。在 add_argument() 中指定了一个新关键字 action ，设置值为 store_true 。这表示如果指定了该选项，就把 args.verbose 的值设为 True ,否则设为 False 。
  • 如果给选项指定值，结果会报错，这是标志量的精髓。
